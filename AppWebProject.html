Website Development Project (Furniture Store)

Associated with COMSATS University Islamabad, for my semester project in web development, I designed and developed a dynamic online furniture store. The project showcased my proficiency in front-end technologies such as HTML, CSS, and JavaScript, creating an engaging and user-friendly interface. The back-end was powered by Node.js and Express, with MongoDB serving as the database.

This platform allowed users to explore a wide range of furniture products, add items to their shopping cart, and seamlessly complete the checkout process, providing a smooth and convenient shopping experience. In addition, I implemented an admin panel that enabled administrators to effortlessly add new products to the store, ensuring the store's catalog remained up-to-date.

Overall, this project demonstrated my comprehensive skill set in web development, encompassing both the user-facing design and the robust back-end functionality required to create a fully functional e-commerce website.
